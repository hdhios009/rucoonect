# RUconnect

Лендинг VPN-сервиса **RUconnect** — быстрый VPN с шифрованием трафика и политикой no-logs.

## Структура

```
.
├── index.html          # главная страница
├── css/
│   └── style.css       # стили
├── js/
│   └── main.js         # интерактив (навигация, FAQ, анимации)
└── images/
    └── logo.jpg        # логотип
```

## Запуск

Откройте `index.html` в браузере — сайт статический, сборка не требуется.

## Хостинг

Работает на GitHub Pages из коробки: **Settings → Pages → Deploy from branch → main / root**.
